# 奇幻动物园预约管理平台中期答辩PPT提纲

## 第1页 封面
- 题目：基于 Spring Boot 的奇幻动物园预约管理系统的设计与实现
- 副标题：中期答辩汇报
- 可补充信息：学生姓名、学号、专业班级、指导教师、答辩日期

## 第2页 课题背景与研究目标
- 随着线上预约和智慧文旅的发展，传统动物园在线服务能力不足，存在信息分散、预约效率低、游客互动弱等问题。
- 本课题拟设计并实现一个面向游客与管理员的动物园预约管理平台。
- 目标包括：
- 实现用户注册登录、动物信息展示、园区地点浏览、门票预约下单等核心业务。
- 实现后台信息管理、社区互动、公告发布、科普资讯和旅游攻略管理。
- 提升动物园信息服务、游客体验和管理效率。

## 第3页 系统总体设计
- 技术架构：前后端分离。
- 前端：Vue3、Vite、Element Plus、Pinia、Vue Router。
- 后端：Spring Boot、MyBatis-Plus、Sa-Token、MySQL。
- 辅助能力：文件上传、二维码支付模拟、定时任务、AI 问答接口预留。
- 系统角色：
- 普通用户：浏览信息、预约购票、发帖互动、收藏点赞、提交举报。
- 管理员：用户管理、内容管理、门票管理、订单管理、互动数据管理。
