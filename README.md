# 奇幻动物园预约管理平台

基于 `Spring Boot + Vue 3` 的前后端分离动物园预约管理系统，面向毕业设计场景开发，集成了动物信息展示、园区地点浏览、门票预约、社区互动、科普资讯、旅游攻略、后台内容管理等功能。

本项目适合作为：

- 毕业设计项目展示
- `Spring Boot + Vue` 前后端分离项目练习
- 动物园 / 景区 / 文旅预约类系统的参考模板

## 项目简介

奇幻动物园预约管理平台围绕“线上展示 + 在线预约 + 社区互动 + 后台管理”四条业务主线展开设计。

系统分为两个主要端：

- 前台用户端：用于游客浏览动物信息、查看园区地点、预约门票、参与社区互动、阅读资讯与攻略。
- 后台管理端：用于管理员维护用户、动物、门票、订单、公告、帖子、评论、资讯和攻略等数据。

## 项目特色

- 前后端分离架构，职责清晰，便于二次开发
- 覆盖“展示、预约、互动、管理”完整业务链路
- 集成 `Sa-Token` 实现登录认证与权限控制
- 支持门票预约、订单管理、库存联动
- 支持帖子、评论、点赞、收藏、举报等社区互动能力
- 支持生态保护资讯与旅游攻略内容管理
- 预留 AI 问答能力，便于扩展智能导览场景
- 前端基于 `Vue 3 + Element Plus`，适合课程设计和毕业设计答辩展示

## 快速开始

### 1. 克隆项目

```bash
git clone <your-repo-url>
cd "SpringBoot + Vue3奇幻动物园预约管理平台"
```

### 2. 初始化数据库

将根目录下的 `copy_animal.sql` 导入 MySQL，创建项目所需数据库与表结构。

如果你使用的是 Navicat / DataGrip / MySQL Workbench，直接执行 SQL 文件即可。

### 3. 修改后端数据库配置

编辑文件：

`backend/src/main/resources/application.yaml`

重点修改以下内容：

```yaml
spring:
  datasource:
    url: jdbc:mysql://127.0.0.1:3306/你的数据库名?serverTimezone=Asia/Shanghai&rewriteBatchedStatements=true
    username: 你的数据库用户名
    password: 你的数据库密码
```

如果你需要使用 AI 问答功能，还需要补充：

```yaml
spring:
  ai:
    deepseek:
      apiKey: 你的 DeepSeek API Key
```

### 4. 启动后端

进入后端目录：

```bash
cd backend
```

安装依赖并启动：

```bash
mvn spring-boot:run
```

或者：

```bash
mvn clean package
java -jar target/base-backend.jar
```

默认后端启动地址：

```text
http://localhost:8080/api
```

### 5. 启动前端

进入前端目录：

```bash
cd frontend
```

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

默认前端启动地址通常为：

```text
http://localhost:5173
```

### 6. 打包前端

```bash
npm run build
```

构建结果输出到：

```text
frontend/dist
```

## 常见问题

### 1. 前端启动后请求失败怎么办？

先确认：

- 后端是否已经启动
- 后端端口是否为 `8080`
- 接口前缀是否为 `/api`
- 前端代理配置是否正确

### 2. 数据库连接失败怎么办？

检查：

- MySQL 服务是否启动
- 数据库名是否正确
- 用户名密码是否正确
- `application.yaml` 中的 IP 是否已改为本机地址

### 3. AI 功能为什么无法使用？

通常是以下原因：

- 没有配置有效的 `DeepSeek API Key`
- 外网不可用
- 调用配额不足

### 4. 为什么有些中文注释显示乱码？

这通常与文件编码、IDE 默认编码或历史拷贝过程有关。建议统一使用：

- `UTF-8`

并检查 IDE 和终端编码设置。
